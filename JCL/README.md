# Some useful JCL code

👍 DISPSYM
   Display symbols in JCL

👍 GENER02
   copy a PS dataset

👍 GENER03
   copy a PS dataset

👍 INTRDR
    writes job cards to a member in a PDS then submits that job through the INTRDR.
    The submitted job writes it's output to a new member in the same PDS.
    Shows how IEBGENER can edit records as it copies them.

👍 PDS2PDSE
   copies a PDS to PDSE.

     You can delete the old PDS or keep it as a backup.

👍 RENAMEDS
     Rename a dataset

👍 WRITEMBR

     WRite some text from the JCL into a PS dataset and into a member
